How To Use

1
Navigate to worldview.earthdata.nasa.gov and add a Brightness Temperature layer
2
Click on the blank space of a layer that has a color palette bar — or click the floating 🌡️ Color Table button
3
A panel shows the full Color ↔ Kelvin correspondence table with swatches. You can export it as CSV or copy to clipboard
